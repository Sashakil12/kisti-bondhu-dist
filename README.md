## About Techtikal ERP

Techtikal ERP is a desktop application built for consumer electronics dealership stores. It covers the full business workflow from inventory to sales, purchasing, customer management, and financial tracking.

**Stack:** React 19 + Ant Design + Apollo Client + Electron (desktop) · Node.js + GraphQL + MongoDB (server)

### Features

- **Brand & Product Management** — Add, update, and manage brands and products with serial number tracking
- **Stock Management** — Stock invoices with serial-level tracking and sync
- **Sales & Invoicing** — Create sales invoices, manage customer dues, print preview with PDF export
- **Customer Management** — Customer profiles, document uploads, due tracking, and aggregated reporting
- **Supplier Management** — Supplier records and deletion workflows
- **Cash Accounts** — Cash account system with split account support and detailed transaction history
- **Transaction References** — Standardized reference system across all transaction types
- **Dynamic Access Control** — Role-based access control (RBAC) configurable per user
- **Import Products** — Bulk product import support
- **Print & PDF** — Configurable PDF headers, orientation, and print preview for invoices and reports
